# 🇨🇳 China Trip Template — Outubro 2026

Roteiro de **22 dias** pela China (com stopover em Istambul), pensado para quem quer fugir do óbvio: foco em **natureza espetacular + cidades vibrantes**, evitando o circuito turístico mais batido.

> Este é um **template** para você adaptar à sua viagem. Datas, voos e reservas são exemplos — troque pelos seus.

## 🗺️ A rota

**Rio → Istambul → Pequim → Zhangjiajie → Furong → Fenghuang → Wangxian Valley → Shanghai → Rio**

| Base | Noites | Destaque |
|------|--------|----------|
| Istambul | 3 | Hagia Sofia, Grand Bazaar, ilha de Büyükada |
| Pequim | 2 | Muralha em Mutianyu (sem Cidade Proibida) |
| Zhangjiajie | 3 | Montanhas Avatar, Tianmen, Glass Bridge |
| Furong | 1 | Vila milenar sobre uma cachoeira |
| Fenghuang | 1 | Vila ribeirinha de palafitas iluminada |
| Wangxian Valley | 1 | Vila de penhasco "网红" |
| Shanghai | 8 | Cidade + bate-voltas (Suzhou, Hangzhou, water towns) |

A grande sacada logística: **Wangxian Valley** fica quase no ponto médio da linha de trem Fenghuang → Shanghai, então parar lá quebra um trajeto de ~8h em duas metades com um destino incrível no meio.

## 🚀 Como usar

1. Abra o `05-CALENDARIO.html` no navegador (celular ou desktop) para ver tudo de relance.
2. Leia `01-VIAGEM.md` e `03-DICAS.md` — são o que mais economizam dor de cabeça.
3. Copie este repositório (botão **Use this template** ou faça um fork) e ajuste para as suas datas e voos.

## ⚠️ Importante para a China

- **Instale VPN no Brasil antes de embarcar** (Google/WhatsApp/Instagram não funcionam sem).
- **WeChat Pay + Alipay** com cartão Visa/Master são essenciais (aceitam estrangeiro desde 2024).
- Compre trens e voos internos no **Trip.com** com 1-2 semanas de antecedência.
